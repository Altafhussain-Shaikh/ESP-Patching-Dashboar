# Patching Dashboard Hub

A lightweight landing dashboard with four sections:

- OS Patching (linked to Power BI)
- DB Patching (linked to Power BI)
- Middleware Patching (upcoming)
- OnPrem Patching (upcoming)

## Run

1. Open `index.html` in your browser.
2. Click OS Patching or DB Patching to open the corresponding Power BI report.

## Notes

- Middleware and OnPrem are intentionally non-clickable and marked as upcoming.
- This project is static HTML/CSS/JS and needs no dependency installation.
